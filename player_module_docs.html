Getting Started

This section guides you on how to include and initialize PlayerModule in your Roblox project. Use the require function to load the module and wrap a player into a proxy object for easy method access.

local PlayerModule = require(path.to.PlayerModule)
local plr = game.Players.LocalPlayer
local myPlayer = PlayerModule.New(plr)

The myPlayer object allows you to call methods directly on the player without repeatedly passing the player reference. This is the foundation for all further interactions.

Character Utilities

These functions provide convenient access to the player's character, humanoid, root part, and movement-related properties.

  • GetCharacter() - Returns the player's character model.
  • GetHumanoid() - Returns the Humanoid instance from the character.
  • GetRootPart() - Returns the HumanoidRootPart, useful for teleportation or positioning.
  • IsAlive() - Checks whether the player is alive based on humanoid health.
  • Respawn() - Respawns the player; can optionally teleport to a custom spawn location.
  • GetLookDirection() - Returns the current look vector of the player.
  • GetVelocity() - Returns the player's movement velocity.
  • GetPosition() - Returns the player's current position as a Vector3.

These functions allow precise control over player interactions, movement detection, and character monitoring.

Team & Player Management

These methods manage team assignments, freezing movement, teleporting, and messaging players.

  • GetTeam() - Returns the player's current team.
  • ChangeTeam(team) - Moves the player to a different team.
  • FreezePlayer(bool) - Freezes or unfreezes a player, affecting walk speed and jump power.
  • SetWalkSpeed(speed) - Directly sets the player's walk speed.
  • SetJumpPower(power) - Sets the humanoid jump power.
  • ResetCharacterAppearance() - Reloads character appearance.
  • TeleportTo(CFrame) - Teleports the player to a specified position or CFrame.
  • SendMessage(text) - Sends a system message to the client (local player).
  • Kick(reason) - Kicks the player from the server (server-side use).

Leaderstats & Currency

Functions for managing leaderstats and game currency, simplifying repetitive tasks in player progression and economy systems.

  • AddLeaderstat(name, value) - Adds a new leaderstat or initializes it if missing.
  • GetLeaderstat(name) - Returns the value of a leaderstat.
  • SetLeaderstat(name, value) - Sets a leaderstat to a specific value.
  • IncrementLeaderstat(name, amount) - Increments a stat by an amount.
  • DecrementLeaderstat(name, amount) - Decrements a stat by an amount.
  • HasGamepass(gamepassId) - Returns true if player owns the gamepass (placeholder for developer implementation).
  • HasAsset(assetId) - Checks asset ownership (placeholder).
  • AwardCurrency(amount, type) - Adds currency (developer-defined).
  • RemoveCurrency(amount, type) - Removes currency (developer-defined).
  • GetCurrency(type) - Retrieves currency amount (developer-defined).

Tools & Inventory

Manage player inventory, equip, unequip, and transfer tools with these methods.

  • GiveTool(tool) - Adds a tool to player's backpack.
  • RemoveTool(name) - Removes a specific tool from backpack and character.
  • RemoveAllTools() - Clears all tools from backpack and character.
  • EquipTool(name) - Equips a tool from backpack.
  • UnequipTool() - Unequips current tool.
  • HasTool(name) - Checks if player owns a tool.
  • CloneTool(tool) - Clones and gives a tool to the player.
  • TransferTool(toPlayer, name) - Moves a tool from this player to another.
  • GetEquippedTool() - Returns currently equipped tool.
  • DropTool(name) - Drops a tool into the workspace.

Events & Animations

Subscribe to player events and manage animations.

  • OnRespawn(callback) - Fires when the player respawns.
  • OnDeath(callback) - Fires on player death.
  • OnTeamChange(callback) - Fires when the team changes.
  • OnToolEquipped(callback) - Fires when a tool is equipped.
  • OnToolUnequipped(callback) - Fires when a tool is unequipped.
  • OnLeaderstatChange(statName, callback) - Fires when a leaderstat changes.
  • BindActionToInput(key, callback) - Binds key input to a callback.
  • PlayAnimation(animId) - Plays an animation on the humanoid.
  • StopAnimation(track) - Stops an animation track.
  • SetRespawnLocation(CFrame) - Sets a custom respawn location.

Server & Client Helpers

Server and client specific methods for managing players.

  • Server.RespawnPlayer(player) - Respawns player on server.
  • Server.Kick(player, reason) - Kicks player from server.
  • Server.GetFromCharacter(character) - Gets player from character (server).
  • Client.GetLocalPlayer() - Returns local player (client-only).
  • Client.GetMouse() - Returns player's mouse object (client-only).
